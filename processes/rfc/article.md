* Keep all RFCs under a single location
* Uniquely identify RFCs
* Use a shared template
	* Editor: Name <email>
	* Status:
	* Required approvers: Name <email> (why), Name <email> (why), ...
	* Approvers: Name <email> (date), Name <email> (date), ...
	* Created on: <date>
	* Updated on: <date>
	* Background
	* Problem
	* Solution
